### Description

Projet étudiant en 3ème année de BUT MMI.
Reproduire à l'identique visuellement une one-page responsive. (Desktop, tablet, mobile)

Page web : "Nintendo, Stardew Valley". 

Le projet est développé avec Next Js.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
