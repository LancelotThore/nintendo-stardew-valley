# 🎮 Stardew Valley – Nintendo Page Reproduction

⭐ Projet universitaire réalisé en **3ᵉ année de BUT MMI (2024 – 2025)**

Ce projet consiste à **reproduire visuellement à l'identique** une page produit du site officiel de Nintendo, en respectant le design et le comportement **responsive** de la page originale.

La page reproduite est celle du jeu **Stardew Valley** sur Nintendo Switch.

# 📱 Aperçu

Reproduction d'une page produit du site Nintendo avec un design responsive adapté aux différents écrans :

* 💻 Desktop
* 📱 Tablet
* 📱 Mobile

Page originale utilisée comme référence :
https://www.nintendo.com/us/store/products/stardew-valley-switch/

# 🎯 Objectif du projet

L'objectif de ce projet était de :

* analyser une **page web existante**
* reproduire fidèlement **le design et la mise en page**
* respecter le **comportement responsive**
* travailler la **précision d'intégration front-end**

Ce type d'exercice permet de développer des compétences en :

* intégration web
* responsive design
* structuration de composants

# ✨ Fonctionnalités

* reproduction fidèle du **design de la page produit**
* **responsive design** pour desktop, tablette et mobile
* intégration des éléments visuels et des sections principales
* structuration en **composants Next.js**

# 🛠️ Technologies utilisées

* **Next.js**
* **React**
* **CSS**

Projet développé en **2024**.
